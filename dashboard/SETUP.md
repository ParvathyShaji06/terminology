# 🚀 Terminology AI Dashboard: Quick Setup

Follow these 3 steps to launch the benchmarker on your local machine.

### ⚡ One-Command Setup (Full Project)
```bash
pip install -r ../requirements.txt && npm install
```

### 1. Prerequisites
Ensure you have **Node.js** installed (v18 or higher).
[Download Node.js](https://nodejs.org/)

### 2. Install & Launch (Copy-Paste)
Run these commands in your terminal:

```bash
# 1. Enter the dashboard directory
cd dashboard

# 2. Install dependencies
npm install

# 3. Start the local server
npm run dev
```

### 3. Open in Browser
Once running, open your browser to:
**👉 [http://localhost:5173/](http://localhost:5173/)**

---

### 🛠️ Key Features for Your Team:
- **API Key**: Enter your Nebius/OpenAI key in the top box.
- **Upload Data**: Use the dashed box to choose your `.xlsx` file.
- **Reset**: Use the Red 'Global System Reset' button near the title to clear everything for a fresh experiment.
- **Exports**: Use the "Export Excel" and "Save Eval View" buttons for your reports.
